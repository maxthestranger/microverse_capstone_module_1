## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

```
node and npm
```

### Setup

```
- clone the repo
- cd into it
- npm install
```

### Install

```
open the html file
```

### Usage

```
navigate through the pages to see different contents
```

### Deployment

```
set up on github pages
```
